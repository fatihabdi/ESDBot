# ESD Bot
this bot created with [discord js](https://discord.js.org/#/)

## Feature
- `.dp` = Create Dropdown Menu Roles
- Welcome.js = create welcoming messages when someone join server

## How to use
- clone this repo
- install dependencies
```bash
npm install
```
- run program
```bash
node .
```
